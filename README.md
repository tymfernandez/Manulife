# RES Project

React + Vite + Tailwind CSS project with separate client and server.

## Structure
- `client/` - React frontend with Vite and Tailwind CSS
- `server/` - Express.js backend

## Setup
```bash
npm run install-all
```

## Development
```bash
npm run dev
```

## Individual Commands
- Client: `npm run client`
- Server: `npm run server`
